要**实时查看网络的具体吞吐量（带宽/流量速率）以及网络“用处”（哪些程序、连接、协议在占用网络）**，取决于你用的是 Windows、macOS、Linux，还是想看路由器层面。下面按场景给你最实用的工具：

------

# 一、最简单：看当前网速（上传 / 下载）

## Windows

### 1. 任务管理器（自带）

快捷键：

```
Ctrl + Shift + Esc
```

进入：

```
性能 → Wi-Fi / 以太网
```

可实时看到：

- 当前发送 Mbps
- 当前接收 Mbps
- 网卡利用率

✅ 最方便，不用安装

------

### 2. Resource Monitor（更详细）

按：

```
Win + R
resmon
```

进入：

```
网络
```

能看到：

- 哪个程序占网速
- TCP连接数
- 实时吞吐量
- 延迟趋势

------

## macOS

### 活动监视器

路径：

```
应用程序 → 实用工具 → 活动监视器
```

查看：

```
网络
```

能看：

- 每个程序发送/接收数据量
- 实时网络压力

------

# 二、专业级：实时看谁在吃带宽（推荐）

## Windows 推荐：GlassWire

优点：

- 图形化实时流量曲线
- 哪个软件在联网一目了然
- 可监控后台偷跑流量

适合普通用户。

------

## Windows/Linux/macOS 通用：Wireshark

专业抓包神器，可查看：

- 每个数据包
- 吞吐量统计
- 哪些协议占流量（HTTP / HTTPS / QUIC / DNS）
- 谁在访问谁

适合技术人员。

------

# 三、Linux 服务器实时吞吐量（最强）

## 1. iftop

像 top 一样看网络：

```
sudo iftop
```

显示：

- 哪个 IP 占带宽
- 上下行实时 Mbps

------

## 2. nload

图形流量条：

```
sudo nload
```

看总吞吐量最舒服。

------

## 3. vnStat

看长期统计：

- 今日流量
- 本月流量
- 历史趋势

------

# 四、查看“网络都用在哪了”

如果你问的是：

> 带宽到底被谁吃掉了？

推荐：

### Windows：

- GlassWire
- Resource Monitor

### Linux：

- iftop
- `ss -tp`
- `netstat -plant`

### 路由器层面：

登录路由器后台，看：

- 每台设备占网速
- 手机/电视/电脑谁在下载

------

# 五、如果你是想看“最大吞吐量”

测试局域网真实吞吐量：

## iperf3

服务器端：

```
iperf3 -s
```

客户端：

```
iperf3 -c IP地址
```

能测：

- 千兆网是否跑满
- Wi-Fi 实际速率
- 丢包率

------

# 六、我直接给你推荐（最实用）

## 普通用户（Windows）

> GlassWire + 任务管理器

## 技术用户

> Wireshark + iperf3

## Linux 服务器

> iftop + nload + vnstat
