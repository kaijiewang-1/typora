## 一、什么是线性回归？

### 1.1 生活中的例子

想象一下，你是一位房产中介，想要根据**房子的大小**来**预测房子能卖多少钱**。

你收集了一些历史数据：
- 50平米的房子 → 卖50万
- 100平米的房子 → 卖100万
- 150平米的房子 → 卖150万

你发现了一个规律：**价格 = 面积 × 1万**

这就是最简单的**线性关系**！

线性回归，就是找到这个"规律"（数学上叫做**拟合一条直线**）。

### 1.2 数学表达

```
预测值 y = θ₀ + θ₁ × x
```

- `x` = 输入特征（比如房子面积）
- `y` = 预测结果（比如房价）
- `θ₀` = 截距（-baseline，相当于起步价）
- `θ₁` = 斜率（-coefficient，每单位x对应的y变化）

## 二、核心概念详解

### 2.1 代价函数（Cost Function）—— 衡量我们做得有多烂

**比喻**：你预测考试成绩和实际成绩的差距

代价函数就是用来衡量**我们的预测有多不准**的指标。

线性回归使用的代价函数叫做**均方误差（MSE）**：

$$J(\theta) = \frac{1}{2m}\sum_{i=1}^{m}(h_\theta(x^{(i)}) - y^{(i)})^2$$

**人话版**：
1. 对于每个数据点，用我的公式预测一个值
2. 用真实值减去预测值，得到误差
3. 把误差平方（消除负号，只看大小）
4. 全部加起来，取平均值

**为什么要平方？**

- 消除正负误差的抵消
- 惩罚大的误差更多（误差越大，平方后更大）

### 2.2 梯度下降（Gradient Descent）—— 不断逼近最优解

**比喻**：你在山顶，想要走到山谷（最低点）

想象你在一座雾山上，只能看到脚下一点点地方。你要走到最低点（山谷），你会怎么做？

**策略**：每一步都往**下坡**的方向走！

这就是梯度下降的核心思想：
1. 从某个起点开始
2. 计算当前位置的"坡度"（梯度）
3. 往坡度相反的方向走一小步
4. 重复，直到走到最低点

**更新公式**：
$$\theta_j = \theta_j - \alpha \frac{\partial}{\partial\theta_j}J(\theta)$$

其中：
- `α` = 学习率（step size，迈的步子大小）
- `α`太大：容易迈过最低点，发散
- `α`太小：走得太慢，半天到不了

### 2.3 特征标准化（Feature Normalization）—— 让数据"公平竞争"

**比喻**：比较苹果和橘子

假设：
- 房子面积：1000-3000 平方英尺
- 卧室数量：1-5 间

这两个数字相差太大！直接比较不公平。

**标准化公式**：
$$x_{normalized} = \frac{x - \mu}{\sigma}$$

- `μ` = 平均值
- `σ` = 标准差

把数据变成：**均值为0，标准差为1** 的分布

### 2.4 正规方程（Normal Equation）—— 直接算出答案

梯度下降是"一步步走"的方法，正规方程是"直接算"的方法：

$$\theta = (X^T X)^{-1} X^T y$$

**优点**：一次就算出最优解，不用迭代
**缺点**：当特征很多时，计算矩阵逆很慢

## 四、实验流程梳理

### 4.1 单变量线性回归（ex1.py）

```
1. 加载数据 → 城市人口和利润
2. 可视化数据 → 画散点图看看分布
3. 添加截距项 → X矩阵前面加一列1
4. 初始化参数 → θ都设为0
5. 计算初始代价 → 用初始参数预测，看误差多大
6. 梯度下降迭代 → 不断更新θ，找到最优解
7. 绘制拟合直线 → 把找到的直线画出来
8. 预测新数据 → 用学到的规律预测新城市
9. 可视化代价函数 → 画3D图和等高线图理解优化过程
```

### 4.2 多变量线性回归（ex1_muti.py）

```
1. 加载数据 → 房子面积、卧室数量、房价
2. 特征标准化 → 让不同特征可以比较
3. 梯度下降法 → 迭代求解
4. 正规方程法 → 直接求解（可选）
5. 预测房价 → 输入新房子信息，预测价格
```

## 五、关键代码解读

### 5.1 代价函数核心

```python
h = X @ theta.T      # 预测值：h(x) = θ₀ + θ₁x
error = h - y        # 误差：真实值 - 预测值
J = (1/(2*m)) * (error.T @ error)  # 代价：误差平方的均值
```

### 5.2 梯度下降核心

```python
h = X @ theta.T              # 预测值
error = h - y                # 误差
theta = theta - (alpha/m) * (error.T @ X)  # 更新参数
```

### 5.3 特征标准化核心

```python
mu = np.mean(X, axis=0)           # 每列的平均值
sigma = np.std(X, axis=0)         # 每列的标准差
X_norm = (X - mu) / sigma         # 标准化
```

### 5.4 正规方程核心

```python
theta = np.linalg.inv(X.T @ X) @ X.T @ y
# θ = (X'X)⁻¹X'y
```

## 六、常见问题解答

### Q1: 学习率α怎么选？
- 太大：可能发散（Cost越来越大）
- 太小：收敛太慢
- 推荐尝试：0.001 → 0.01 → 0.1 → 1

### Q2: 怎么判断收敛了？
- 看代价函数值是否不再下降
- 或者固定迭代次数（比如1500次）

### Q3: 为什么要加一列1？
- 为了计算截距项θ₀
- X变成 [1, x]，这样 θ₀×1 + θ₁×x 包含截距

## 七、总结

线性回归的核心就是：

1. **假设**：数据之间存在线性关系
2. **代价**：用均方误差衡量预测好坏
3. **优化**：用梯度下降找到使代价最小的参数
4. **预测**：用学到的参数预测新数据

这就是机器学习最基本的思想：**从数据中学习规律，然后用规律做预测**！
