### 📊 前端三剑客核心概览

| 技术领域                | 核心职责                         | 关键概念/组件概览                                    |
| :---------------------- | :------------------------------- | :--------------------------------------------------- |
| **HTML (结构层)**       | 定义网页的**结构和内容**         | 文档结构、语义化标签、文本标签、多媒体、表单         |

### 超文本标记语言 (HTML)

HTML是网页的骨架，负责定义文档的结构和内容。

#### 一、文档结构与语义化

一个标准的HTML5文档基本结构如下：

html

下载

复制

预览

```
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>网页标题</title>
</head>
<body>
    <!-- 页面可见内容 -->
</body>
</html>
```

- `<!DOCTYPE html>`声明了文档类型为HTML5

  

- `<meta charset="UTF-8">`定义了字符编码，确保正确显示中文

  

- `<meta name="viewport">`是响应式设计的关键，使页面能自适应不同设备的屏幕

  

- **语义化标签**是HTML5的重要特性，它们不仅划分区域，更明确了内容的角色，如 `<header>`、`<nav>`、`<main>`、`<article>`、`<section>`、`<aside>`、`<footer>`。使用它们能显著提升网页的可访问性（对屏幕阅读器等辅助技术友好）和SEO（搜索引擎优化）效果

  

#### 二、常用内容标签与属性

- **文本与媒体**：

  - **标题**：`<h1>`到 `<h6>`，重要性依次递减

    

  - **段落**：`<p>`

    

  - **图片**：`<img src="image.jpg" alt="图片描述">`，`alt`属性在图片无法加载时显示，对可访问性至关重要

    

  - **音视频**：HTML5引入了 `<audio>`和 `<video>`标签，使嵌入多媒体内容更简单

    

- **列表**：

  - **无序列表**`<ul>`：列表项 `<li>`默认以实心圆点标示

    

  - **有序列表**`<ol>`：列表项 `<li>`默认以数字标示

    

- **超链接**：`<a href="https://www.example.com" target="_blank">链接文本</a>`。`target="_blank"`实现在新标签页打开

  

- **全局属性**：

  - `id`：为元素指定**唯一**标识

    

  - `class`：为元素指定一个或多个类名，主要用于CSS和JavaScript选择

    

#### 三、表单

表单用于收集用户输入，是与服务器交互的重要组件

html

- **初级验证**：HTML5提供了 `required`（必填）、`pattern`（正则表达式验证格式）等属性进行基本的表单验证

  

### 层叠样式表 (CSS)

CSS控制页面的表现，包括布局、颜色、字体等。

#### 一、基础语法与引入方式

CSS规则由**选择器**和**声明块**组成

css

复制

```
p { /* p 是选择器 */
    color: blue; /* color: blue; 是一条声明 */
    font-size: 16px; /* 属性: 值; */
}
```

- **引入方式**：

  - **外部样式表**（推荐）：`<link rel="stylesheet" href="styles.css">`

    

  - **内部样式表**：在HTML的`<head>`内使用`<style>`标签书写

    

  - **内联样式**（不推荐）：直接在元素上使用`style`属性，如`<p style="color: red;">`

    

#### 二、核心概念

- **选择器**：用于选中要添加样式的元素。

  - 基础选择器：`*`（通配符）、`tag`（标签）、`.class`（类）、`#id`（ID）

    

  - 组合选择器：`div p`（后代选择器）、`div > p`（子选择器）

    

  - 伪类/伪元素：`:hover`（鼠标悬停状态）、`::before`（在元素前插入内容）

    

- **盒模型**：每个元素都被视为一个矩形盒子，从内到外由**内容**、**内边距**、**边框**、**外边距**构成

  

  - `width`和`height`默认只设置内容区的宽高。`box-sizing: border-box;`是一个非常重要的属性，它会让元素的`width`和`height`包含内边距和边框，使得尺寸计算更直观

    

- **层叠与优先级**：当多个规则作用于同一元素时，优先级规则为：`!important`> 内联样式 > ID选择器 > 类/伪类/属性选择器 > 标签选择器。 specificity相同时，后定义的样式生效

  

#### 三、布局技术

- **Flexbox 弹性布局**：适用于**一维**布局（沿直线排列）。

  css

  复制

  ```
  .container {
      display: flex;
      justify-content: center; /* 主轴对齐方式 */
      align-items: center; /* 交叉轴对齐方式 */
  }
  ```

- **Grid 网格布局**：适用于**二维**布局（行和列）。

  css

  复制

  ```
  .container {
      display: grid;
      grid-template-columns: 1fr 1fr 1fr; /* 定义三列等宽 */
      gap: 20px; /* 网格间隙 */
  }
  ```

- **定位**：`position`属性。

  - `relative`：相对自身原位置进行偏移。

  - `absolute`：相对于最近的非`static`定位的父元素进行定位。

  - `fixed`：相对于浏览器视口进行定位，不随滚动条移动。

  - `sticky`：在视口内表现为`relative`，滚动超出时表现为`fixed`（如吸顶效果）

    

#### 四、响应式设计与动画

- **响应式设计**：使用**媒体查询**针对不同屏幕尺寸应用不同样式

  css

  复制

  ```
  @media (max-width: 768px) {
      body { font-size: 14px; }
      .sidebar { display: none; }
  }
  ```

- **过渡与动画**：

  - **过渡**：`transition: all 0.3s ease-in-out;`使属性变化更平滑

    

  - **动画**：使用`@keyframes`定义更复杂的动画序列

    css

    复制

    ```
    @keyframes slidein {
        from { transform: translateX(-100%); }
        to { transform: translateX(0); }
    }
    .box { animation: slidein 1s ease-in-out; }
    ```

### JavaScript

JavaScript 是一种动态编程语言，用于为网页添加交互逻辑



#### 一、语言基础

- **变量声明**：

  javascript

  下载

  复制

  运行

  ```
  let age = 25; // 块级作用域，值可变
  const PI = 3.14; // 块级作用域，常量，不可重新赋值
  var oldName = "John"; // 函数作用域，不推荐使用

- **数据类型**：分为基本类型和对象。

  - **基本类型**：`String`, `Number`, `Boolean`, `undefined`, `null`, `Symbol`

    

  - **对象**：`Object`, `Array`, `Function`, `Date`等

    

- **函数**：封装可重复使用的代码块。

  javascript

  下载

  复制

  运行

  ```
  // 函数声明
  function greet(name) {
      return 'Hello, ' + name;

  // 箭头函数 (ES6+)

#### 四、异步 JavaScript

处理需要等待的操作（如网络请求），而不阻塞页面。

- **Promise 与 async/await**：是现代JavaScript处理异步的主要方式。

  javascript

  下载

  复制

  运行

  ```
  // 使用Fetch API发起网络请求 (返回一个Promise)
  fetch('https://api.example.com/data')
      .then(response => response.json()) // 解析JSON数据
      .then(data => console.log(data)) // 处理数据
      .catch(error => console.error('出错：', error)); // 错误处理
  
  // 使用async/await语法（更清晰）
  async function fetchData() {
      try {
          const response = await fetch('https://api.example.com/data');
          const data = await response.json();
          console.log(data);
      } catch (error) {
          console.error('出错：', error);
      }
  }
  ```
