## 1. 前言

### 1.1 产品简介

欢迎使用散养家禽数智化养殖综合管理平台！这是一款专门为养殖企业精心打造的智慧农业移动管理应用，旨在帮助养殖户实现养殖场的数字化、智能化管理，提高养殖效率，降低运营成本。

本平台集成了养殖场管理、环境监控、设备运维、工单处理、企业协同等核心功能，通过简洁直观的移动端界面，让管理人员能够随时随地掌控养殖场的一切。无论是查看实时环境数据、管理智能设备、分配工作任务，还是了解企业运营状况，只需在手机上轻轻一点，即可完成所有操作。

我们致力于为每一位用户提供优质的使用体验。如果您在使用过程中遇到任何问题或有好的建议，请随时联系我们的技术支持团队。

### 1.2 主要功能一览

本平台提供了丰富多样的功能模块，全面覆盖养殖场的日常管理需求：

- **账号管理功能**：支持用户注册、登录、密码找回等基础服务，还特别提供了游客登录模式，方便新用户快速体验平台功能。企业用户还可以创建自己的企业并邀请员工加入，实现团队协同办公。

- **主控制台功能**：是整个应用的首页，以仪表盘的形式直观展示养殖场的关键数据。您可以在这里快速查看当前的温度、湿度等环境参数，了解养殖场数量、设备数量、预警数量等统计信息，并通过卡片入口快速进入各个功能模块。

- **设备管理功能**：帮助您集中管理养殖场的各类智能设备，包括摄像头、传感器和控制器三种类型。您可以随时查看设备在线状态、添加新设备、查看设备详情，甚至可以观看摄像头的实时视频直播。

- **农场管理功能**：支持多养殖场的集中管理，您可以创建新的养殖场、查看养殖场详情、管理养殖场负责人，所有养殖场地信息尽在掌握。

- **工单管理功能**：实现了工作任务的流程化、规范化管理。从工单创建到派发、接单、签到、完成，每个环节都有完整的记录和跟踪，让工作任务有迹可循。

- **环境监控功能**：实时展示养殖场的环境参数，包括温度、湿度、光照、土壤湿度、二氧化碳浓度、风速等关键指标，并支持历史数据查询和异常预警功能。

- **企业管理功能**：用于管理企业的基本信息和组织架构，包括企业信息查看与编辑、企业成员管理等。

- **个人中心功能**：允许用户管理个人信息、修改密码、切换所属企业等。

- **消息通知功能**：用于接收和展示系统推送的各种通知信息，确保用户不会错过重要信息。

- **管理员功能**：为企业管理员提供高级管理功能，包括用户管理、企业配置等。

## 2. 使用前准备

### 2.1 系统要求

为确保应用能够正常运行，您的设备需要满足以下基本要求：

- **操作系统要求**：Android 7.0（API 24）或更高版本的Android系统。部分新功能可能需要更高版本的Android系统才能正常运行，建议您保持系统更新以获得最佳使用体验。

- **网络环境要求**：应用需要连接互联网才能正常使用各项功能。建议使用WiFi网络以获得更稳定的数据传输体验。在使用移动数据时，请注意您的流量套餐情况。

- **存储空间要求**：应用安装包大小约为几十兆字节，日常使用中会产生一定的缓存数据。建议保留至少200MB的可用存储空间，以确保应用能够正常安装和运行。

- **权限要求**：应用需要以下系统权限才能正常工作——网络访问权限（用于与服务器通信）、网络状态检查权限（用于判断网络连接状态）。这些权限在首次启动时会自动申请，您只需允许即可。

### 2.2 应用下载与安装

本应用的安装包可以通过以下渠道获取：

- **官方下载渠道**：访问应用官方网站或扫描官方提供的二维码即可下载最新版本的应用安装包。请注意识别官方渠道，避免下载到假冒版本。

- **应用商店分发**：应用可能会在各大Android应用商店上架，您可以在应用商店中搜索"数智化养殖软件平台"或相关关键词进行下载安装。

安装步骤如下：首先在您的Android设备上找到下载的安装包文件，点击该文件开始安装。如果系统提示"禁止安装未知来源应用"，请前往设备的"设置→安全"页面，允许"未知来源应用"安装。安装过程中请耐心等待，安装完成后即可打开应用。

### 2.3 首次启动与账号准备

首次启动应用时，系统会引导您进行简单的初始化设置。按照屏幕提示完成操作后，您将进入登录页面。

如果您已经拥有账号，可以直接输入手机号或用户名和密码进行登录。系统支持"记住密码"功能，登录成功后会自动保存用户名，下次登录时只需输入密码即可。

如果您是新用户，可以点击"注册账号"按钮进行用户注册。注册过程中需要填写手机号、设置密码等基本信息。注册完成后，使用注册的账号即可登录。

如果您想先体验一下应用的功能，可以点击"游客登录"按钮。系统为您预置了游客账号（手机号：13800138007，密码：138007），登录后可以浏览大部分功能模块。注意，游客账号的部分功能可能受限，且数据不会长期保存。

### 2.4 企业关联

登录系统后，如果您的账号还未关联任何企业，系统会引导您进入企业选择或创建页面。您可以选择"加入已有企业"，输入企业名称或邀请码申请加入；也可以选择"创建新企业"，按照提示填写企业信息后创建属于自己的企业。

加入企业或创建企业后，您就可以正常使用平台的所有功能了。如果您的账号关联了多个企业，可以在个人中心中切换当前所属企业。

## 3. 核心功能使用指南

### 3.1 账号登录与安全

#### 3.1.1 用户登录

登录是进入系统的第一步，正确登录后才能访问各项功能。登录页面提供了清晰明了的输入框和按钮，您只需按照以下步骤操作即可：

首先，在登录页面的用户名输入框中填写您的手机号或用户名。系统支持两种登录方式，如果您记得注册时使用的手机号，请直接填写手机号；如果您使用的是用户名登录，请填写用户名。输入框有格式提示，请注意查看。

接下来，在密码输入框中填写您的登录密码。密码区分大小写，请确保输入正确。如果不小心输错，可以点击右侧的清空按钮重新输入。

信息填写完成后，点击"登录"按钮提交登录请求。系统会验证您的账号信息，验证成功后自动跳转到主控制台页面。如果账号或密码错误，系统会显示错误提示，请仔细核对后重新输入。

为提升使用便利性，系统提供了"记住密码"选项。勾选此选项后，系统会自动保存您的用户名，下次登录时只需输入密码即可。但请注意，在公共设备上使用时请勿勾选此选项，以保护账号安全。

#### 3.1.2 游客登录

如果您只是想先了解一下应用的功能，或者暂时没有注册账号，可以选择游客登录模式。点击登录页面中的"游客登录"按钮，系统会自动使用预设的游客账号进行登录。

游客登录的操作流程与普通登录完全一致，只需点击按钮即可完成。登录成功后，您可以浏览主控制台、查看设备列表、了解农场信息等，但部分敏感功能（如创建工单、管理企业等）可能无法使用。

需要特别说明的是，游客账号是为了方便用户体验而设置的，其数据不会长期保存，也不保证可用性。如果您需要长期使用平台功能，建议注册一个正式账号。

#### 3.1.3 账号注册

新用户需要先注册账号才能使用平台功能。点击登录页面的"注册账号"按钮，即可进入注册页面。注册过程需要填写以下信息：

手机号是重要的身份标识，请填写您真实有效的手机号。系统会向该手机号发送验证码进行验证，因此请确保手机号可以正常接收短信。手机号填写格式为11位数字，以1开头。

设置密码时，请按照页面提示的密码强度要求操作。建议使用包含字母、数字的组合密码，长度不少于6位。密码是保护您账号安全的重要手段，请妥善保管，不要告诉他人。

确认密码需要再次输入您设置的密码，以确保两次输入一致。完成以上信息填写后，点击"注册"按钮提交注册申请。注册成功后，系统会自动跳转到登录页面，您可以使用刚注册的账号进行登录。

#### 3.1.4 密码找回

如果您忘记了登录密码，可以通过以下方式找回：点击登录页面的"忘记密码"按钮，系统会提示您联系管理员重置密码。这是因为系统采用手机号绑定的方式，管理员可以验证您的身份后为您重置密码。

联系管理员时，请提供您的注册手机号和身份信息，管理员核实后会为您重置密码。重置后的密码可能是系统默认密码，登录后请及时修改为自己熟悉的密码。

### 3.2 主控制台

主控制台是登录后的首页，也称为仪表盘，是整个应用的导航中心。从这里，您可以快速了解养殖场的整体状况，也可以快速进入各个功能模块。

#### 3.2.1 页面结构

主控制台页面主要分为以下几个区域：

- **顶部区域**：显示当前的温度和湿度等关键环境参数。这些数据来自部署在养殖场内的传感器，会自动实时更新，让您随时了解当前的环境状况。

- **欢迎区域**：显示欢迎语和当前登录用户的信息。

- **统计区域**：展示企业的核心统计数据，包括养殖场数量（显示为"X个养殖场"）、设备数量（显示为"X台设备"）、预警数量（显示为"X个预警"）等。这些数据来自后端服务器的实时统计，能够帮助您快速了解企业运营状况。

- **功能卡片区域**：以卡片形式展示各个功能模块的入口。卡片上标有功能名称和图标，点击卡片即可进入相应功能。默认展示的功能卡片包括：设备管理、农场管理、企业信息、个人资料、工单管理、环境监控、管理员入口等。

- **底部导航栏**：提供快捷导航功能，底部导航栏包含五个菜单项，分别是主页（仪表盘）、设备、农场、工单、企业，点击相应图标即可切换到对应模块。底部导航栏在大多数页面都会显示，方便您快速切换。

#### 3.2.2 数据更新

主控制台的数据会自动进行更新。页面首次加载时会显示加载动画，数据获取完成后自动展示。每次进入主控制台页面时，系统都会自动发起数据请求，获取最新的统计信息。

此外，系统还设置了自动刷新机制，每隔30秒会自动更新一次数据，确保您看到的信息是最新的。如果您想手动刷新数据，可以下拉页面触发刷新操作。

### 3.3 设备管理

设备管理是系统的核心功能之一，用于集中管理养殖场的各类智能设备。通过这个模块，您可以查看所有设备的运行状态、添加新设备、管理设备信息等。

#### 3.3.1 设备列表

进入设备管理页面后，首先看到的是设备列表。列表中展示了所有已添加的设备，每个设备项都包含以下信息：

- **设备名称**：您为设备设置的名称，便于识别。

- **设备类型**：设备的具体类型，分为摄像头、传感器、控制器三种。摄像头图标为监控样式，传感器图标为感应样式，控制器图标为开关样式。

- **所属站点**：设备所绑定的管理站点位置。

- **在线状态**：显示设备当前是否在线。在线状态用绿色圆点表示，离线状态用灰色圆点表示。

如果设备较多，列表会自动进行分页。当您滑动到列表底部时，系统会自动加载下一页数据。页面顶部显示设备统计信息，包括设备总数、在线设备数、离线设备数，一目了然。

#### 3.3.2 添加设备

点击页面上的"添加设备"按钮，会弹出创建设备对话框。在对话框中需要填写以下信息：

- **设备名称**：为设备起一个容易识别的名称，例如"1号鸡舍摄像头"、"温度传感器A"等。名称长度建议控制在20个字符以内。

- **所属站点**：从下拉列表中选择设备所属的管理站点。如果还没有创建站点，需要先在农场管理中创建站点。

- **MAC地址**：设备的物理地址，每台设备的MAC地址是唯一的。正确填写MAC地址是确保设备能够被系统识别的关键。

- **设备类型**：从下拉列表中选择设备类型。选择"摄像头"时，还需要填写推流名称；选择"传感器"或"控制器"时则不需要。

- **推流名称**（仅摄像头设备填写）：用于视频直播的推流标识，需要根据实际配置填写。推流名称用于建立视频流连接，错误会导致视频无法播放。

填写完所有信息后，点击"创建"按钮提交。创建成功后，新设备会立即出现在设备列表中。如果创建失败，系统会显示错误提示，请检查信息填写是否正确后重试。

#### 3.3.3 查看设备详情

在设备列表中点击任意设备项，即可进入该设备的详情页面。详情页面展示了设备的完整信息，包括设备ID、设备名称、设备类型、MAC地址、所属站点、在线状态、创建时间等。

对于摄像头设备，详情页面还会显示"观看直播"按钮。点击该按钮可以进入直播页面，观看该摄像头的实时视频画面。观看直播需要网络环境稳定，建议在WiFi环境下观看以获得最佳体验。

#### 3.3.4 删除设备

对于不再需要的设备，您可以将其删除。在设备列表中，长按要删除的设备项，或者点击设备详情页面中的删除按钮，系统会弹出确认删除对话框。

删除操作是不可逆的，删除后设备的所有数据将被清除，且无法恢复。因此，在确认删除之前，请务必确认该设备确实不再需要。确认无误后，点击"删除"按钮执行删除操作。

### 3.4 农场管理

农场管理模块用于管理养殖企业的养殖场信息。如果您负责管理多个养殖场，这个模块将帮助您高效地管理所有养殖场地。

#### 3.4.1 农场列表

进入农场管理页面后，映入眼帘的是农场列表。列表中展示了企业下所有的养殖场，每个养殖场项显示以下信息：

- **农场名称**：养殖场的名称，是识别养殖场的主要标识。

- **地址**：养殖场的详细地址，便于定位和导航。

- **负责人**：负责该养殖场管理的人员姓名。

- **类型**：养殖场的类型，如养殖场、种植园、混合农场等。

- **运营状态**：养殖场的当前运营状态，如正常运营、暂停运营、已关闭等。

#### 3.4.2 创建农场

点击页面上的"添加农场"按钮，弹出创建农场对话框。在对话框中需要填写以下信息：

- **农场名称**：为养殖场起一个具有辨识度的名称，例如"第一养殖场"、"山地散养基地"等。

- **农场地址**：填写养殖场的详细地址信息，包括省市县、乡镇、村庄等。

- **负责人**：从企业员工列表中选择该养殖场的负责人。被选中的负责人将负责该养殖场的日常管理工作。

信息填写完成后，点击"创建"按钮提交。创建成功后，新农场会出现在农场列表中。

#### 3.4.3 农场详情

点击农场列表中的任意农场项，可以查看该养殖场的详细信息。详情页面展示了农场的完整信息，还可以查看该农场下的站点分布和设备分布情况。

#### 3.4.4 删除农场

对于不再需要的农场，您可以将其删除。在农场列表中，长按要删除的农场项，系统会弹出确认删除对话框。删除操作同样是不可逆的，请谨慎操作。

### 3.5 工单管理

工单管理模块实现了养殖工作任务的流程化、规范化管理。无论是日常巡检、设备维护，还是突发情况处理，都可以通过工单系统进行管理。

#### 3.5.1 工单列表

进入工单管理页面后，首先看到的是工单列表。列表上方有两个标签页："全部"和"我的"。

- **全部标签**：展示所有工单，无论创建者或执行者是谁，都可以在这里看到。这是管理员查看所有工作任务情况的窗口。

- **我的标签**：只展示与当前用户相关的工单，包括我创建的工单、我执行的工单等。这是普通员工查看个人任务的窗口。

每个工单项显示工单标题、状态、创建时间、执行人等信息。工单状态用不同的颜色标识：待接单（灰色）、已接单（蓝色）、进行中（黄色）、已完成（绿色），便于快速识别工单进度。

#### 3.5.2 创建工单

点击页面上的"创建工单"按钮，进入工单创建页面。在创建页面中需要填写以下信息：

- **工单标题**：简要描述工单内容，例如"检查1号鸡舍摄像头"、"清理3号饲料仓库"等。标题应简明扼要，便于识别。

- **工单内容**：详细描述工单的具体要求，包括工作内容、工作地点、注意事项等。内容描述越详细，执行人越容易理解任务要求。

- **执行人**：从企业员工列表中选择任务的执行人。被选中的执行人将收到工单派发通知。

填写完以上信息后，点击"提交"按钮创建工单。工单创建成功后，会自动派发给选定的执行人，执行人可以在"我的"工单中看到该工单。

#### 3.5.3 执行工单

作为执行人，当您收到派发的工单后，可以在"我的"标签中找到该工单。工单的处理流程如下：

- **接单**：点击工单项进入详情页面，点击"接单"按钮确认接单。接单表示您开始执行该任务。

- **签到**：到达工作现场后，点击"签到"按钮进行签到。签到会记录您到达现场的时间和位置，确保您确实到现场执行了任务。

- **完成任务**：工作完成后，点击"完成"按钮提交完成申请。在提交时需要填写完成情况说明，描述完成的具体内容。

工单完成后，创建者可以对执行情况进行评价，帮助改进工作质量。

#### 3.5.4 工单详情

点击任意工单项，可以查看该工单的详细信息和执行进度。详情页面展示工单的完整内容、执行流程、状态变更历史等信息。管理员可以在这里查看工单的执行进度，监督任务完成情况。

### 3.6 环境监控

环境监控模块是保障养殖质量的重要功能。通过这个模块，您可以实时查看养殖场的环境参数，及时发现异常情况。

#### 3.6.1 实时数据

进入环境监控页面后，首先看到的是当前的环境参数数据。系统展示了以下关键指标：

- **温度**：当前环境温度，单位为摄氏度（°C）。温度是影响禽类生长的重要因素，需要保持在适宜范围内。

- **湿度**：当前环境相对湿度，单位为百分比（%）。湿度过高或过低都会影响禽类健康。

- **光照**：当前光照强度，单位为勒克斯（lux）。光照影响禽类的采食和产蛋。

- **土壤湿度**：地表或养殖区域的土壤含水量，单位为百分比。主要用于监测地面潮湿程度。

- **二氧化碳**：当前二氧化碳浓度，单位为ppm。二氧化碳浓度过高会导致禽类缺氧。

- **风速**：当前空气流动速度，单位为米/秒。通风状况对禽类健康有重要影响。

这些数据来自部署在养殖场内的传感器，会自动实时更新。如果某个数据项显示"--"或异常值，可能是对应传感器未连接或故障，请检查设备状态。

#### 3.6.2 历史查询

页面上有"今日"、"本周"、"本月"三个时间筛选按钮。点击不同按钮可以查看不同时间段的历史数据。

- **今日数据**：查看当天的环境参数变化趋势。

- **本周数据**：查看本周（7天）内的环境参数变化。

- **本月数据**：查看本月（30天）内的环境参数变化。

通过查看历史数据，您可以分析环境参数的变化规律，发现潜在问题。例如，如果某天的温度异常升高，可能需要检查通风设备是否正常工作。

#### 3.6.3 预警信息

页面下方展示了环境异常预警信息。当环境参数超出正常范围时，系统会自动生成预警记录。预警信息包括预警类型、预警内容、产生时间等。

如果收到预警，请及时关注并采取相应措施。例如，温度过高时需要加强通风，湿度过大时需要除湿等。

### 3.7 企业管理

企业管理模块用于管理企业的基本信息和组织架构。这个模块主要面向企业管理员，普通员工可能只有查看权限。

#### 3.7.1 企业信息

企业信息页面展示了当前所属企业的基本信息，包括：

- **企业名称**：企业的官方名称。

- **企业地址**：企业的经营地址。

- **联系人**：企业负责人或对接人的姓名。

- **联系电话**：企业的联系电话。

这些信息可以在企业信息页面中进行编辑修改。修改后的信息会同步到企业所有成员的客户端。

#### 3.7.2 企业成员

企业成员页面展示了企业下所有员工的列表。列表中包括成员的姓名、手机号、状态等信息。管理员可以在这里管理企业成员，包括查看成员信息、邀请新成员等。

### 3.8 个人中心

个人中心是用户管理个人信息的专属空间。进入方式有两种：点击主控制台的"个人资料"卡片，或点击底部导航栏最右侧的企业图标后选择"个人资料"。

#### 3.8.1 个人信息

个人信息页面展示了当前登录用户的基本信息，包括用户名、手机号、所属企业等。这些信息来自注册时填写的数据，部分信息可能无法直接修改。

#### 3.8.2 修改密码

为保护账号安全，建议您定期修改密码。点击"修改密码"功能，输入当前密码和新密码后即可完成密码修改。新密码需要与确认密码一致，且需要满足密码强度要求。

#### 3.8.3 切换企业

如果您同时加入了多个企业，可以在"切换企业"功能中选择当前要操作的企业。切换后，页面显示的数据将切换为所选企业的数据。

### 3.9 消息通知

消息通知模块用于接收和展示系统推送的各种通知消息，确保用户不会错过重要信息。

#### 3.9.1 通知列表

进入消息通知页面后，可以看到所有通知的列表。通知按时间倒序排列，最新的通知显示在最上方。

每条通知显示通知标题、内容摘要、接收时间等信息。未读通知会有特殊标记（如小红点或颜色高亮），便于识别。

#### 3.9.2 通知详情

点击任意通知项，可以查看该通知的完整内容。详细内容页面展示了通知的标题、完整内容、发送时间等信息。

查看通知后，该通知会自动标记为已读状态。未读标记消失，表示您已经阅读了该通知。

### 3.10 管理员功能

管理员功能为企业管理者提供高级管理权限。只有具有管理员身份的用户才能访问此模块。

#### 3.10.1 权限验证

进入管理员模块时，系统会自动验证您的管理员权限。如果没有权限，系统会提示"无企业权限"，您将无法进入管理员页面。

#### 3.10.2 用户管理

管理员可以查看和管理企业下的所有用户，包括查看用户列表、用户状态等信息。

// JavaScript/Axios 调用示例
const login = async (phone, password) => {
  try {
    const response = await axios.post('http://124.71.97.178:8080/user/login', {
      phone: '13800138000',
      password: '123456'
    });
    
    if (response.data.code === 0) {
      const { satoken, userId, enterpriseId } = response.data.data;
      // 保存Token到本地存储
      localStorage.setItem('satoken', satoken.tokenValue);

      console.error('登录失败:', response.data.message);

    console.error('网络错误:', error);

};

## 5. 常见问题与解决方案

### 5.1 登录问题

#### 5.1.1 登录失败，提示账号或密码错误

请仔细检查输入的用户名或手机号是否正确，注意大小写和空格。如果忘记密码，请联系管理员重置。确认账号状态是否正常，如账号被禁用请联系管理员处理。尝试使用游客账号登录，验证是否为账号问题。

#### 5.1.2 登录时提示网络异常

请检查您的设备是否已连接网络，可以尝试打开网页验证网络是否正常。如果使用WiFi，请确保WiFi连接正常且可以访问互联网。如果使用移动数据，请确认移动数据已开启且流量充足。尝试切换网络环境，如从WiFi切换到移动数据，或反之。可能是服务器端暂时不可用，稍后再试。

#### 5.1.3 应用闪退或无法启动

请确认设备系统版本是否满足要求（Android 7.0及以上）。尝试重启设备后再次打开应用。清除应用缓存后重新打开：进入手机设置，找到应用管理，选择本应用，点击清除缓存。尝试卸载后重新安装应用。

### 5.2 数据问题

#### 5.2.1 数据加载失败或显示为空

请检查网络连接是否正常，尝试刷新页面获取最新数据。如果确认网络正常但仍无法加载，可能是后端服务器暂时不可用，请稍后再试。对于刚创建的企业，可能还没有添加任何数据，这是正常现象，请先添加设备或农场数据。

#### 5.2.2 数据显示异常或不准确

如果发现数据显示异常，首先确认数据来源设备是否正常工作。传感器设备可能存在精度问题或故障，建议定期检查设备状态。如果异常数据持续出现，请联系技术支持人员进行处理。

#### 5.2.3 历史数据查询不到

历史数据查询需要系统提前存储相应时长的数据。如果查询不到历史数据，可能是因为数据存储周期已过，或者对应的传感器设备当时未在线。请确认查询的时间范围内设备是否在线。

### 5.3 设备问题

#### 5.3.1 设备显示离线状态

设备离线可能由以下原因导致：设备电源未开启或断电，请检查设备供电情况；设备网络连接中断，请检查设备的网络配置；设备距离路由器太远信号弱，请调整设备位置或增加信号放大器；设备本身故障，请联系设备供应商维修。

#### 5.3.2 视频直播无法播放

视频直播无法播放的常见原因包括：网络带宽不足，请确保网络环境稳定且带宽充足；推流地址配置错误，请检查设备推流名称是否正确；设备不支持直播功能，请确认设备类型为摄像头；直播服务暂时不可用，请稍后再试。

#### 5.3.3 无法添加设备

无法添加设备的可能原因包括：设备MAC地址已存在系统中，请使用唯一的MAC地址；所属站点不存在或已删除，请先创建站点；网络连接异常，请检查网络后重试；设备类型选择错误，请根据实际设备类型选择。

### 5.4 工单问题

#### 5.4.1 无法创建工单

请检查是否已加入企业，只有企业成员才能创建工单。确认必填信息是否完整，工单标题和内容不能为空。检查网络连接是否正常。

#### 5.4.2 工单无法派发

请确认执行人是否为企业成员，只有企业成员才能被指定为执行人。检查执行人账号状态是否正常。

#### 5.4.3 无法接单或完成任务

请确认当前账号是否为该工单的执行人。只有执行人才能进行接单和完成操作。检查工单当前状态是否允许该操作，例如已完成状态的工单无法再次接单。

### 5.5 其他问题

#### 5.5.1 页面显示错位或显示异常

这可能是由于屏幕适配问题。请尝试退出应用后重新进入。如果问题持续，可能是应用版本与系统版本不兼容，请尝试更新应用到最新版本。

#### 5.5.2 应用运行缓慢

请检查设备存储空间是否充足，空间不足可能导致应用运行缓慢。关闭后台运行的其他应用程序，释放系统资源。尝试清除应用缓存数据。考虑升级设备系统版本或更换性能更好的设备。

#### 5.5.3 收到陌生企业邀请

如果您收到来自陌生企业的邀请，请谨慎处理。不要轻易同意加入来历不明的企业。如有疑问，可以联系平台客服进行咨询。

## 7. 附录

### 7.1 名词解释

为帮助您更好地理解本平台的相关概念，以下是一些常用术语的解释：

- **养殖场**：指进行养殖生产的具体地点，可以是一个鸡舍、一个放养区域或一个完整的养殖基地。

- **站点**：养殖场内的具体管理区域，例如1号鸡舍、2号鸡舍、饲料仓库等。设备需要绑定到具体站点进行管理。

- **设备**：连接到系统的智能硬件设备，包括用于视频监控的摄像头、用于环境数据采集的传感器、用于设备控制的控制器等。

- **工单**：需要执行的工作任务单据，记录了任务内容、执行人、时间要求等信息。

- **企业**：使用本平台进行管理的养殖企业或组织，一个企业可以包含多个养殖场和多个员工。

### 7.2 操作流程图

以下是主要操作的简要流程图，供您参考：

**登录流程**：

```
打开应用 → 输入账号密码 → 点击登录 → 验证成功进入主控制台
```

**添加设备流程**：

```
进入设备管理 → 点击添加设备 → 填写设备信息 → 点击创建 → 设备添加成功
```

**创建工单流程**：

```
进入工单管理 → 点击创建工单 → 填写工单内容 → 选择执行人 → 提交工单 → 工单派发成功
```

**执行工单流程**：

```
在"我的"工单中找到任务 → 点击接单 → 到达现场签到 → 完成任务 → 提交完成
```

### 7.3 安全建议

为保护您的账号和企业的数据安全，建议您遵循以下安全措施：

- **密码安全**：使用强度较高的密码，定期更换密码，不要使用过于简单的密码。不要将密码告诉他人，尤其是在未经确认的情况下。

- **账号保管**：不要在公共设备上保存登录信息，使用公共设备后记得退出登录。如果账号发生异常，及时联系管理员。

- **企业安全**：谨慎处理企业邀请，只有确认身份后才加入陌生企业。企业管理员应定期检查成员列表，及时清理离职人员账号。

- **设备安全**：定期检查设备在线状态，及时处理离线设备。保护好设备MAC地址等敏感信息，防止被恶意利用。
