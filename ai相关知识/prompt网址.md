如果你是想找 **大模型提示词（Prompt）灵感库、模板库、实战案例网站**，下面这些比较值得收藏。我按用途给你分类，很多都能免费使用。

------

## 一、最实用的 Prompt 模板网站（直接抄来用）

### 1. FlowGPT

**网址：** https://flowgpt.com

适合：ChatGPT / Claude / Midjourney 提示词模板

优点：

- 海量用户分享 Prompt
- 分类清晰（写作、编程、学习、营销）
- 可直接复制使用
- 很多“爆款提示词”

适合新手入门。

------

### 2. PromptHero

**网址：** https://prompthero.com

适合：Midjourney、Stable Diffusion、ChatGPT

优点：

- 图像生成提示词特别强
- 看别人怎么写关键词
- AI绘画必备

如果你做设计，这个很好用。

------

### 3. AIPRM

**网址：** https://www.aiprm.com

适合：ChatGPT网页端用户

优点：

- 装插件后直接在 ChatGPT 里调用模板
- SEO、写作、营销、代码很多

办公党常用。

------

## 二、中文 Prompt 网站（更适合国内用户）

### 4. AI Short

**网址：** https://www.aishort.top

优点：

- 中文提示词库
- 各类场景：论文、翻译、运营、Excel、写代码
- 一键复制

很适合你直接拿来提升效率。

------

### 5. PromptPerfect

**网址：** https://promptperfect.jina.ai

适合：

- 你写了一个 prompt，但感觉效果一般
- 它帮你自动优化提示词

相当于“Prompt润色器”。

------

## 三、专业学习 Prompt Engineering（进阶）

### 6. OpenAI Prompting Guide

官方/高质量教程类

适合学习：

- Zero-shot
- Few-shot
- Chain of Thought
- Role Prompting
- Structured Output

如果你想真正学会写 prompt，这类最重要。

------

### 7. Learn Prompting

**网址：** https://learnprompting.org

优点：

- 系统课程
- 免费学习 Prompt Engineering
- 从初级到高级

非常推荐。
